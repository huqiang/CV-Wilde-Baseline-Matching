NOTE:

  • Ths program is not tested on Python3
  • SIFT feature detector cannot be used in Windows Platform
  • Dont use same image as both QUERY_IMAGE and TRAINED_IMAGE
    #### General Idea:
  • Detectect keypoints
  • Extract descriptors
  • Find matches
  • Keep good matches
  • Perform geo-verification using RANSAC
  • Further filter out no-so-good matchs based on found foundamental matrix or homography
